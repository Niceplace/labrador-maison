# Opencode server & web ui

Starts the opencode server with web UI, configured behind Traefik

# Access

## Browser 

[https://opencode.thinkcenter.dev](https://opencode.thinkcenter.dev)

## Terminal UI

You will need the opencode password for this, it's defined in Komodo secrets

`opencode attach http://opencode.thinkcenter.dev:4096 --password $OPENCODE_SERVER_PASSWORD`
